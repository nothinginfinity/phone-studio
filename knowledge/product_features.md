## Phone Studio (PWA)

### Core Features

<!-- List current live features -->

- Screenshot → OCR → LLM processing
- Multi-provider LLM support (Groq, DeepSeek, Mistral, Claude, GPT-4, Gemini, etc.)
- Voice memo import and semantic indexing
- Batch photo processing
- Local Review Queue (approve/export drafts)
- Content creation wizard (Instagram, TikTok, LinkedIn, Twitter, Blog, Email)
- Creator Profile (saved brand context)
- Semantic compression (300–500 byte photo metadata)
- BYOK (Bring Your Own API Key — stored on-device only)
- PWA — installable on iPhone home screen
