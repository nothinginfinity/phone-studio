📱 Voice Memo Export Setup

Set up an iPhone Shortcut so Voice Memos exports audio and transcript files together.

1

Create iOS Shortcut

Open Shortcuts on iPhone and create a new shortcut using the export recipe below.

2

Enable iCloud Drive

Make sure iCloud Drive and Shortcuts syncing are enabled on the iPhone.

  1. Settings → [Your Name] → iCloud
  2. Toggle on iCloud Drive
  3. Toggle on Shortcuts for iCloud sync
3

Grant Files Access

The first import may ask Phone Studio to access Files or an exported folder. Allow access so matching audio and transcript files can be read.

4

Record → Export → Import

  1. Record in Voice Memos and let iOS transcribe it
  2. Run the Shortcut: “Export to Phone Studio”
  3. Save audio to `iCloud Drive/PhoneStudio/audio/`
  4. Save transcript to `iCloud Drive/PhoneStudio/transcripts/`
  5. In Phone Studio, tap `Check iCloud Drive` and then `Import All`

Export to Phone Studio Shortcut

In Shortcuts, create a new shortcut and add these actions in order:

1
Ask for Voice Memo

Let the user select a memo from the Voice Memos library.

2
Get File

Get the audio file for the selected voice memo.

3
Get Details of Voice Memo

Capture metadata like date and duration.

4
Get Transcript

Use the iOS transcript text from Voice Memos and save it as plain text.

5
Generate Timestamp Filename

Format the current date as `MEMO_[date_time]` so audio and transcript filenames match.

6
Save Audio to iCloud Drive

Folder: `iCloud Drive/PhoneStudio/audio/` with filename `[timestamp].m4a`.

7
Save Transcript to iCloud Drive

Folder: `iCloud Drive/PhoneStudio/transcripts/` with filename `[timestamp].txt`.

8
Show Success Message

Confirm export so the user knows the files are ready for Phone Studio.

Check iCloud for New Memos

After running the Shortcut, import matched audio and transcript files here. If direct iCloud folder access is limited, choose the exported files manually from Files.
